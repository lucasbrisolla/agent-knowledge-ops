# Knowledge Synthesis Example

Exemplo curto de evidence bundle para mostrar como múltiplas fontes podem alimentar uma decisão de promoção.

## Cenário

Um agente de operações precisa decidir se deve criar um checklist para mudança de processo.

As fontes dizem coisas parecidas, mas com autoridade diferente:

- uma reunião recente registra a decisão
- um runbook antigo descreve o processo anterior
- uma conversa de chat mostra risco operacional recorrente

## Uso

Leia `evidence-bundle.md` como exemplo de artefato intermediário.

Ele não é conhecimento canônico. Ele mostra a ponte entre fontes e promoção.
